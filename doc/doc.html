Deployment

You have some ways to deploy the static website

Visual Studio Code

If you are using Visual Studio Code, you can install the extension Live Server.

After that, you only need to open the index.html file and right click. Click at "Open with live server" and instantly it will open your browser and show you the site.

XAMPP

Open you XAMPP application, start your apache service, and navigate with your browser to the root of the apache server in the htdocs folder. It's important to put the project in thhe htdocs folder, inside the xampp installation folder to avoid some server connection problems.
